# Flying Squirrel Portfolio

Personal portfolio for [flyingsquirrel0419](https://github.com/flyingsquirrel0419), focused on shipped backend systems, developer tools, runtimes, and Team WICKED production work.

The project explorer covers layercache, date-light, unlimited-search, claude-occ, RuJa, and Iodine V2 with problem, implementation, evidence, and stack details for each project.

## Run locally

```bash
npm install
npm run dev
```

Production checks:

```bash
npm run lint
npm run build
```
